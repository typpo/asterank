Exoplanet API

Overview

Asterank is provides a database containing data from the NASA Exoplanet Archive. It is a simple way to quickly apply constraints to a set of over 800 confirmed exoplanets.

The database currently runs on mongodb and queries must adhere to mongo's json format for a 'find' operation. Information is updated nightly.

Format

Requests are of the form:

http://asterank.com/api/exoplanets?query={query}&limit={limit}

Sample Request

{% raw %} This request returns 10 exoplanets with estimated temperatures between 290 and 320 Kelvin.

/api/kepler?query={"TPLANET":{"$lt":320,"$gt":290}}&limit=10

{% endraw %}
